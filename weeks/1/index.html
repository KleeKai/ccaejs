Hello, JavaScript.

Let's talk about the history later. Here is what you practically need to know right now:

  • JavaScript is the language of web browsers. Using it, you can manipulate elements contained within the Document Object Model
  • JavaScript was created in 10 days in 1995 by Brendan Eich at Netscape; it shows. However, the community has adopted the good parts and matured them
  • JavaScript is a loosely typed language. White space does not matter. Comments are done C-style (// to comment a line, /* */ for longer comments)
  • If you are comfortable with a C-like language, the basic syntax will feel immediately familiar

Deep down, JavaScript has more in common with Lisp and Scheme than with Java. It is Lisp in C's clothing. This makes JavaScript a remarkably powerful language.

Douglas Crockford, Javascript: The Good Parts
