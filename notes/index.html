OSX Basics

Just turn the thing on

  • In case the machines aren't on ahead of time, the power button is in the back
  • You can log on/off using the Apple logo in the top left hand of your screen

Application Equivalence

  • Finder is like Explorer in Windows or Nautilus (et. al.) in Linux
  • Terminal is like DOS in Windows or the Terminal in Linux
  • Spotlight is like the Start bar in Windows 7 or Gnome Do in Linux
  • Our text editor for the course will be Sublime Text 2

Hotkeys

Cmd here mean the Command key. It may be labeled on your keyboard as just the Apple logo. Or if you are using a keyboard intended for Windows, the Command key is the Windows key.

  • Copy - Cmd + C
  • Paste - Cmd + V
  • New Directory - Cmd + Shift + N
  • Switch Applications - Cmd + Tab
  • Switch Between Application Windows - Cmd + ~
  • Opening Spotlight - Cmd + Space

Using the Terminal

The terminal can be opened by hitting Cmd + Space to open Spotlight and then typing Terminal. The first result will be Terminal.app, so you can hit enter to open the application.

Commands

  • List files - ls
  • Change directories - cd [destination]
  • Show current directory - pwd
  • Copy - cp [source] [destination]
  • Move - mv [source] [destination]
  • Delete - rm [file]
  • Delete directory - rm -r [directory]
  • Make directory - mkdir [name]

Conditionals

Basic syntax

// Basic syntax
if (condition) {
  doSomething();
} else if (otherCondition) {
  doSomethingElse();
} else {
  fallBack();
}

Comparison Operators

// Equality and inequality
if ("hello" == "hi")      // False
if ("hello" != "hi")      // True

if (1 == 1)               // True
if (1 != 1)               // False

if (1 == "1")             // True
if (1 != "1")             // False


// Exact equality and inequality
if (1 === "1")            // False
if (1 !== "1")            // True

if (1 === 1)              // True
if (1 !== 1)              // False

if ("hello" === "hello")  // True
if ("hello" !== "hello")  // False


// Greater and less than
if (2 < 4)                // True
if (2 > 4)                // False

if (2 >= 2)               // True
if (2 <= 2)               // True

Logical Operators

// AND
if (age > 12 && age < 20) {
  alert("You are a teenager.");
}

// OR
if (myFood == "apple" || myFood == "orange") {
  alert("This is maybe a fruit. I should have created a more exhaustive list, perhaps.")
}

// NOT
if (!true) {
  // That's the same as false. We'll never get here.
}

// All together now
if ((this || that) && !(here || there))
